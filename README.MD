\## Περιγραφή

Το \*\*StreetFoodGo\*\* είναι μια διαδικτυακή εφαρμογή που επιτρέπει στους χρήστες να παραγγέλνουν φαγητό από καταστήματα. Υποστηρίζει ρόλους πελάτη και ιδιοκτήτη καταστήματος, διαχείριση παραγγελιών και προβολή ισοτιμίας νομισμάτων σε πραγματικό χρόνο.



Η εφαρμογή υλοποιήθηκε με \*\*Spring Boot\*\* και ακολουθεί αρχιτεκτονική υπηρεσιών με χρήση REST APIs.



\## Τεχνολογίες

\* \*\*Γλώσσα:\*\* Java 17

\* \*\*Framework:\*\* Spring Boot

\* \*\*Βάση Δεδομένων:\*\* H2 (In-memory)

\* \*\*Build Tool:\*\* Maven

\* \*\*Template Engine:\*\* Thymeleaf

\* \*\*API Documentation:\*\* OpenAPI / Swagger UI



\## Προαπαιτούμενα

Για να εκτελεστεί η εφαρμογή, χρειάζεται να είναι εγκατεστημένα:

1\.  \*\*Java JDK 17\*\* (ή νεότερο).

2\.  \*\*Maven\*\*.



\## Οδηγίες Εκτέλεσης



\### Μέθοδος 1: Μέσω Τερματικού (Command Line)

1\.  Ανοίξτε τερματικό (CMD/Terminal) στον φάκελο του project.

2\.  Εκτελέστε την εντολή:

&nbsp;   \* Windows: `mvnw.cmd spring-boot:run`

&nbsp;   \* Linux/Mac: `./mvnw spring-boot:run`



\### Μέθοδος 2: Μέσω IDE (IntelliJ IDEA)

1\.  Ανοίξτε τον φάκελο του project στο IntelliJ ("Open as Project").

2\.  Βρείτε το αρχείο `src/main/java/.../StreetFoodGoApplication.java`.

3\.  Κάντε δεξί κλικ και επιλέξτε \*\*Run 'StreetFoodGoApplication'\*\*.



Η εφαρμογή θα ξεκινήσει στη θύρα \*\*8080\*\*.

\## Εξωτερικές Υπηρεσίες

Η εφαρμογή επικοινωνεί με το `exchangerate-api.com` για την ανάκτηση της ισοτιμίας EUR/USD, η οποία εμφανίζεται στη σελίδα του κάθε καταστήματος.

Η εφαρμογή επίσης εκθέτει ένα πλήρως τεκμηριωμένο REST API που επιτρέπει την διαλειτουργικότητα με εξωτερικά συστήματα, υποστηρίζοντας την ανάκτηση δεδομένων καταστημάτων και τη δημιουργία παραγγελιών μέσω αιτημάτων JSON.
